  • Both null and undefined are special values in JavaScript that represent the absence of a meaningful value.
    undefined is a primitive value in JavaScript that represents the absence of a value that should exist but hasn't been assigned yet. A variable declared but not initialized explicitly is assigned the value undefined by.
    Use cases of undefined: To represent the absence of a value in a variable that hasn't been assigned.

    null is a special value that represents the intentional absence of any value or object.
    Use cases of null:
    To explicitly indicate that a variable or object property has no value.

  • REST API, or Representational State Transfer Application Programming Interface, is a design pattern for building web services that provide a standardized way for different software applications to communicate with each other over the internet. REST is an architectural style that focuses on using a set of well-defined principles to create scalable and maintainable APIs.
